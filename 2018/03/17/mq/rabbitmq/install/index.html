前言

用过消息队列的应该知道，它的作用有：异步处理、应用解耦、流量削峰、日志处理等等。

之前用过的 ActiveMQ 消息队列，但是新东家使用的是 RabbitMQ，所以接下来会着重学习下 RabbitMQ，并结合一些实际应用场景开发一些小 Demo，争取能在入职前对其有一个基本的掌握。
