  • еӨҡзәҝзЁӢ

й”ҒжҳҜз”ЁжқҘжҺ§еҲ¶еӨҡзәҝзЁӢи®ҝй—®е…ұдә«иө„жәҗзҡ„дёҖз§Қж–№ејҸпјҢдёҖиҲ¬жғ…еҶөдёӢпјҢй”ҒеҸҜд»ҘйҳІжӯўеӨҡдёӘзәҝеҗҢж—¶и®ҝй—®е…ұдә«иө„жәҗпјҢдҪҶжңүдәӣй”ҒеҸҜд»Ҙе…Ғи®ёеӨҡдёӘзәҝзЁӢе№¶еҸ‘ең°и®ҝй—®е…ұдә«иө„жәҗпјҢеҰӮиҜ»еҶҷй”Ғдёӯзҡ„иҜ»й”ҒгҖӮ

Lock жҺҘеҸЈ

дёҖиҲ¬жғ…еҶөдёӢпјҢжҲ‘д»¬еёёз”Ё synchronized е…ій”®еӯ—жқҘе®һзҺ°й”Ғзҡ„еҠҹиғҪпјҢеүҚйқўжҸҗеҲ° synchronize е…ій”®еӯ—жҳҜеҲ©з”ЁеҜ№иұЎзӣ‘и§ҶеҷЁжқҘе®һзҺ°еҗҢжӯҘпјҢsynchronizeиҺ·еҸ–й”Ғе’ҢйҮҠж”ҫй”ҒеҫҲеӣәе®ҡпјҢе°ұжҳҜе…ҲиҺ·еҸ–еҶҚйҮҠж”ҫпјҢиҷҪз„¶з®ҖдҫҝдәҶеҗҢжӯҘеҢ–з®ЎзҗҶпјҢдҪҶжҳҜдёҚеӨҹзҒөжҙ»ж–№дҫҝгҖӮжҜ”еҰӮжҲ‘д»¬иҰҒе®һзҺ°иҝҷз§ҚйңҖжұӮпјҡиҺ·еҸ–й”Ғ A еҗҺпјҢеҶҚеҺ»иҺ·еҫ—й”Ғ BпјҢиҺ·еҸ–й”Ғ B еҗҺйҮҠж”ҫ AпјҢеҶҚеҺ»иҺ·еҫ—й”Ғ CпјҢиҺ·еҫ—й”Ғ C еҗҺйҮҠж”ҫ BпјҢеҶҚеҺ»иҺ·еҫ—й”Ғ DвҖҰвҖҰеңЁиҝҷз§ҚеңәжҷҜдёӢпјҢдҪҝз”Ё synchronize е…ій”®еӯ—е°ұдёҚеӨӘе®№жҳ“е®һзҺ°дәҶпјҢjdk1.5еҗҺж–°еўһдәҶ Lock жҺҘеҸЈпјҢдҪҚдәҺ concurrent еҢ…дёӢгҖӮ

Lock жҸҗдҫӣдәҶдёҺ synchronized е…ій”®еӯ—еҗҢж ·зҡ„еҠҹиғҪпјҢеҸӘжҳҜеңЁдҪҝз”Ёж—¶йңҖиҰҒжҳҫзӨәең°иҺ·еҸ–й”ҒгҖҒйҮҠж”ҫй”ҒпјҢиҖҢ synchronize е…ій”®еӯ—дёҚйңҖиҰҒпјҢдҪҶжҳҜ Lock дјҡжӣҙзҒөжҙ»пјҢе®ғжҸҗеҚҮдәҶй”Ғзҡ„иҺ·еҸ–гҖҒйҮҠж”ҫзҡ„ж“ҚдҪңжҖ§пјҢжҸҗдҫӣдәҶй”ҒеҸҜдёӯж–ӯиҺ·еҸ–й”ҒгҖҒи¶…ж—¶иҺ·еҸ–й”Ғзҡ„зӯү synchronize е…ій”®еӯ—дёҚе…·еӨҮзҡ„еҠҹиғҪгҖӮ

жіЁпјҡsynchronized е…ій”®еӯ—йҡҸзқҖдёҚж–ӯең°дјҳеҢ–пјҢжҖ§иғҪи¶ҠжқҘи¶Ҡй«ҳпјҢjdk8зҡ„ ConcurrentHashMap жҠӣејғдәҶ ReentrantLock иҖҢдҪҝз”Ё synchronized

жіЁж„ҸпјҡйҮҠж”ҫй”ҒиҰҒеңЁ finally д»Јз Ғеқ—йҮҢпјҢдҝқиҜҒй”ҒжңҖз»ҲиғҪиў«йҮҠж”ҫпјҢдёҚиҰҒе°ҶиҺ·еҸ–й”Ғзҡ„иҝҮзЁӢж”ҫеңЁ try д»Јз Ғеқ—дёӯпјҢеӣ дёәеҰӮжһңиҺ·еҸ–й”Ғж—¶еҸ‘з”ҹдәҶејӮеёёпјҢејӮеёёжҠӣеҮәзҡ„еҗҢж—¶пјҢд№ҹдјҡеҜјиҮҙй”Ғж— ж•…йҮҠж”ҫпјҢtyr д»Јз Ғеқ—йҮҢеҠ й”ҒеӨұиҙҘпјҢдјҡиө° finally еҶҚеҺ»йҮҠж”ҫй”ҒдјҡжҠӣеҮәвҖқIllegalMonitorStateExceptionвҖқејӮеёёпјҢиҝҷдёҚжҳҜжҲ‘д»¬жғіиҰҒзҡ„гҖӮиҖҢж”ҫеҲ°try д»Јз Ғеқ—еӨ–пјҢеңЁиҺ·еҸ–й”ҒеӨұиҙҘеҗҺзәҝзЁӢйҖҖеҮәе№¶жҺ’йҳҹйҮҚж–°еҺ»иҺ·еҫ—й”ҒгҖӮ

е°қиҜ•йқһйҳ»еЎһең°иҺ·еҸ–й”ҒпјҡеҪ“еүҚзәҝзЁӢе°қиҜ•иҺ·еҸ–й”ҒпјҢеҰӮжһңиҝҷдёҖж—¶еҲ»й”ҒжІЎжңүиў«е…¶д»–зәҝзЁӢиҺ·еҸ–еҲ°пјҢеҲҷжҲҗеҠҹиҺ·еҸ–е№¶жҢҒжңүй”ҒгҖӮ

и¶…ж—¶иҺ·еҸ–й”ҒпјҡеңЁжҢҮе®ҡзҡ„жҲӘжӯўж—¶й—ҙд№ӢеүҚиҺ·еҸ–й”ҒпјҢеҰӮжһңеңЁжҢҮе®ҡзҡ„ж—¶й—ҙеҶ…д»ҚжңӘиҺ·еҸ–й”ҒеҲҷиҝ”еӣһпјҢйҒҝе…Қжӯ»й”ҒгҖӮ

з®Җз§° AQSпјҢеңЁжҹҘзңӢ Lock жҺҘеҸЈе…ёеһӢзҡ„зҡ„еӯҗзұ» ReentrantLock е’Ң ReentrantReadWriteLock жәҗз ҒеҗҺеҸ‘зҺ°иҝҷдәӣзұ»зҡ„е®һзҺ°йғҪжҳҜйҖҡиҝҮиҒҡеҗҲ AQS зҡ„еӯҗзұ»жқҘе®ҢжҲҗзәҝзЁӢи®ҝй—®жҺ§еҲ¶зҡ„гҖӮ

AQSпјҡйҳҹеҲ—еҗҢжӯҘеҷЁпјҢжҳҜз”ЁжқҘжһ„е»әй”Ғзҡ„еҹәзЎҖжЎҶжһ¶пјҢдҪҝз”ЁдёҖдёӘ int зұ»еһӢзҡ„жҲҗе‘ҳеҸҳйҮҸиЎЁзӨәеҗҢжӯҘзҠ¶жҖҒпјҢйҖҡиҝҮеҶ…зҪ®зҡ„ FIFO йҳҹеҲ—жқҘе®ҢжҲҗиө„жәҗиҺ·еҸ–зәҝзЁӢжҺ’йҳҹзҡ„е·ҘдҪңпјҲж‘ҳиҮӘгҖҠJavaе№¶еҸ‘зј–зЁӢзҡ„иүәжңҜгҖӢпјү

AQSжҳҜдёҖдёӘжҠҪиұЎзұ»пјҢжүҖд»ҘйңҖиҰҒеӯҗзұ»з»§жүҝе®ғжқҘдҪҝз”ЁпјҢеӯҗзұ»жҺЁиҚҗиў«е®ҡд№үдёәиҮӘе®ҡд№үеҗҢжӯҘз»„д»¶зҡ„йқҷжҖҒеҶ…йғЁзұ»пјҲеҰӮReentrantLock зҡ„ SyncпјүпјҢеҗҢжӯҘеҷЁж”ҜжҢҒзӢ¬еҚ ејҸе’Ңе…ұдә«ејҸең°иҺ·еҸ–й”ҒгҖӮ

AQS йҮҮз”ЁдәҶвҖңжЁЎжқҝж–№жі•вҖқи®ҫи®ЎжЁЎејҸпјҢе°ҶдёҖдәӣеә•еұӮж“ҚдҪңе°ҒиЈ…жҲҗжЁЎжқҝж–№жі•пјҢз®ҖеҢ–дәҶй”Ғзҡ„е®һзҺ°гҖӮ
